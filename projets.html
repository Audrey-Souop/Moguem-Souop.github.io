FPGA-Accelerated Map Inflation for Robotics

Spring 2026 · FPGA Acceleration · Robotics Navigation

Designed a specialized hardware accelerator to speed up map inflation, a key operation in robotic navigation for collision avoidance in dynamic environments. The project focused on achieving deterministic, low-latency performance for safe human-robot collaboration.

  • Designed a Sparse Patch Systolic Array (SPSA) architecture for deterministic map inflation
  • Built a hybrid pipeline combining software-based detection with hardware-based inflation logic
  • Implemented and optimized the RTL using AMD Vivado and Vitis
  • Mapped multiplication operations to DSP primitives to reduce LUT utilization
  • Achieved high-frequency and low-power implementation on Spartan-7 FPGA

FPGA-Accelerated 1-D Time-Domain Convolution

Fall 2025 · ZedBoard · Hardware-Software Co-Design

Designed and implemented a custom hardware accelerator for high-speed 1-D convolution on FPGA, using parallel computation and efficient buffering to outperform a microprocessor-based solution.

  • Developed a parallel datapath with 128 multipliers and an adder tree
  • Built a custom DMA-like interface for streaming data from emulated DRAM
  • Implemented clock domain crossing techniques between PS and PL
  • Designed a sliding-window buffer to sustain high-throughput data delivery
  • Integrated the accelerator into a custom SoC with memory-mapped control logic

Synthetic Dataset Generation for Robotic Arms

Spring–Summer 2025 · ROS 2 · Gazebo · Computer Vision
